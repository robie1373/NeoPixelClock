Neopixelclock

Download this project as a .zip file Download this project as a tar.gz file

I’ve had a sweet chunk o’ granite for about 9 years that I’ve always wanted to turn into a clock.

I had to drill a hole in the bottom to get the control and charging wires out.

Adafruit sells a fun, individually addressable RGB LED ring. The microcontroller is a Flora. Timekeeping is handled by a Chronodot RTC. The big cap is there to protect the Neopixels as per Adafruits best practices. I don't think it's really needed in this project since the power source is either LiPo batter or the charger. Better safe than sorry.

There's a 5-way button on the back for future use.

Power is supplied by a LiPo
